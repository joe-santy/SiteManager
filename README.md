##### SiteManager  

## CLI-based website creation and management  

##### Websites created with this package consist of static HTML/CSS/JS files hosted on an Express server.  Templates are used to generate these pages.  

### Technologies  
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
  <li>Bootstrap</li>
  <li>jQuery</li>
  <li>Node.js</li>
  <li>Express</li>
  <li>BASH</li>
</ul>
